# FilmsParser

1. Go to directory with .dll file                                                 
2. Enter and execute dotnet MoviesParser.dll -category=0

-category:
default - tv,
0 - tv,
1 - movie
-platforms:
exmple - -platforms=1,2,3,4
1,2,3,4 - id`s of platforms
-count:
integer of count elements to parse,
default - infinity.
